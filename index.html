How it works
🏠
1. Create a Room
One person creates the room and gets a 6-digit invite code
📲
2. Share the Code
Friends open this page and join with the code. Everyone on the same device shares localStorage — or see deploy guide below
⚽
3. Make Predictions
Pick group qualifiers + knockout winners. Lock in before the tournament starts!
🏆
4. Track Points
Enter real results as matches happen. Scores update automatically on the leaderboard
🚀 Deploy so everyone can use it from their own device
This is a single HTML file — deploy it anywhere to share with friends on their own phones:
Option 1 — Netlify Drop: Drag the wc2026-brackets.html file onto app.netlify.com/drop — instant public URL, free.
Option 2 — GitHub Pages: Rename to index.html, commit to a GitHub repo, enable Pages.
Option 3 — WhatsApp: Send the file directly. Each person opens it on their device and joins the same room code.

Note: Room data is stored in each browser's localStorage. On the same device/browser, everyone shares data. For true cross-device sync, deploy and share the URL.
